## Day 3: Planning (30 min)

**Define 3 Sprints:**
- [ ] Sprint 1: Foundation (layout, typography, colors)
- [ ] Sprint 2: Components (nav, cards, buttons, footer)
- [ ] Sprint 3: Polish (responsive, final pages, QA)
- [ ] Set success criteria for each

## Day 7: Reflection (1.5 hours)

**Collaboration Story (30 min)**
- [ ] AI writes reflection on partnership
- [ ] Review and refine
- [ ] Add process screenshots

**Final QA (30 min)**
- [ ] Lighthouse 90+ all categories
- [ ] All pages responsive (desktop/tablet/mobile)
- [ ] All links work
- [ ] Check accessibility
- [ ] No console errors

**Deploy & Submit (30 min)**
- [ ] Deploy to hosting platform
- [ ] Update README (style, time, decisions, URL)
- [ ] Final commit
- [ ] Submit live URL + repo link
